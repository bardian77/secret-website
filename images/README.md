# Photos

Drop your pictures of you and SP in this folder.

By default the site looks for:

- `us-1.jpg`
- `us-2.jpg`
- `us-3.jpg`

You can add more (or use different names) — just update the `PHOTOS` list at
the top of `script.js` to match. Any photo that isn't found is simply skipped,
so a missing file never breaks the page.
